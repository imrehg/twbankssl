Notes

  • Grade: grade derived from lowest graded endpoint, also shown the 60-day evolution of the grade
  • Endpoint grade: grade received by SSL Test (from F to A+) for each detectred endpoint, or X if it couldn't be graded for any reason.
  • RC4 support: using RC4 is now prohibited by the IETF; using it seems to be an automatic PCI compliance failure, and there exists a password recovery attack againts it.
  • POODLE: indicates whether the server is vulnerable to the POODLE attack.
  • FREAK: indicates whether the server is vulnerable to the FREAK (Factoring RSA Export Keys) exploit, meaning it supports 512-bit key exchange.
  • Forward Secrecy: indicates whether there's forward secrecy available, at least with modern browsers.
  • Warnings: does the site has any warnings that affect its rating?
